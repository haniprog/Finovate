# Finovate

Finovate is a simple web-based financial management system designed to help users efficiently manage their personal finances. The platform enables users to monitor income, expenses, and financial records through an intuitive and user-friendly interface while providing administrators with tools to manage system data.

# Problem It Solves

Managing personal finances manually can be time-consuming and prone to errors. Many individuals struggle to keep track of their income and expenses, making it difficult to monitor spending habits and make informed financial decisions. Finovate addresses this problem by providing a centralized platform where users can securely record, organize, and monitor their financial transactions, helping them gain better control of their finances.

# How to Run

# Prerequisites

* XAMPP
* Web browser

# Installation

1. Clone or download this repository.
2. Move the project folder into the `htdocs` directory of XAMPP.
3. Start **Apache** and **MySQL** in XAMPP.
4. Open **phpMyAdmin** and import the SQL file from the `database` folder.
5. Update the database configuration file if necessary.
6. Open the application in your browser.

```text
http://localhost/Finovate/
```

### Admin Access

```text
http://localhost/Finovate/admin/login.php
```
